（3）IPv6（注：有些楼层不能用）
IPv6是Internet Protocol Version 6的缩写，是用于替代现行版本IP协议（IPv4）的下一代IP协议。IPv6正处在不断发展和完善的过程中，它支持更多的服务类型和提供更优质的服务。
IPv6地址设置非常简单，在WindowsXP/Windows2003操作系统下只需要在开始->运行->输入ipv6 install然后执行就可以了；在Windows Vista和Windows 7操作系统下是默认开启的。
这样安装IPv6协议栈后，对原IPv4网络的使用不产生任何影响，此时，用户计算机及整个校园网在一个物理网络上，同时运行 IPv4 和 IPv6 双栈、同时支持两种协议的数据传输。
如何判断IPv6是否配置成功？
在开始->运行->输入cmd确定->输入ipconfig后应该能够看到一个以2001:为前缀的IPv6地址，然后在浏览器中输入http://v6.fudan.edu.cn/，如果能够访问则说明IPv6已经配置成功了。否则有可能是所在区域的网络接入服务有问题，请咨询管理员。

通过IPv6免费访问IPv4网站
在浏览器地址栏中输入网站地址，在最后加上.sixxs.org，就可以访问IPv4的网站，但对于要求使用cookie的网站可能无法正常登陆。

（6）网络故障排查
如果按照上述设置好之后，还不能上网，那么按照以下步骤进行错误排查：
ping 127.0.0.1 若通则网卡正常
ping 自己的IP 若通则网线正常
ping 网关 若通则网关正常
ping 校内正常情况下可以访问的IP地址，比如202.120.224.5，若不通则网关可能有问题
ping www.fudan.edu.cn 若不能解析域名则DNS有问题
若以上皆不通，则在命令提示符中中执行route -f，然后重新连接网络，按上述顺序进行测试
特殊情况:
(a)病毒干扰 
(b)相关硬件故障，可以重装下网卡驱动，或者换根网线、网口
